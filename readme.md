## Задание 1:  Таблица истинности

У нас на входе есть 2 операнда — **A** и **B**. Они могут принимать значения **0** или **1**. 

Используя  PHP, HTML и CSS, продемонстрировать таблицу истинности для  операций  

* *отрицание*, 
* *логическое ИЛИ*,
* *логическое И*,
* *исключающее ИЛИ*.
